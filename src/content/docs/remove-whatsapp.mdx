> Perhatian : menghapus nomor WhatsApp akan mengakibatkan seluruh data seperti database, data campaign, data bot dan lainnya pada nomor WhatsApp tersebut hilang! Lakukan backup data / download seluruh data pada nomor yang ingin dihapus terlebih dahulu sebelum melakukan penghapusan nomor WhatsApp dari Dripsender

Silahkan Anda login ke akun Dripsender atau klik saja di sini https://portal.dripsender.id/whatsapp. Lalu ikuti langkah-langkah berikut :
