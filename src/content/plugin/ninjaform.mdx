## Pengenalan

Dripsender NinjaForm Plugin adalah solusi integrasi yang memungkinkan Anda menghubungkan form NinjaForm dengan layanan Dripsender melalui webhook. Plugin ini memungkinkan pengiriman data form secara otomatis ke sistem Dripsender untuk pemrosesan lebih lanjut seperti campaign WhatsApp, list building, dan automation marketing.

> Link download plugin Dripsender-NinjaForm ada di akhir artikel ini.

## Instalasi

### 1. Install Dripsender NinjaForm Plugin
Pastikan plugin NinjaForm sudah terinstall dan aktif di WordPress Anda.
1. Unduh file zip plugin Dripsender NinjaForm dari link yang disediakan
2. Di dashboard WordPress, navigasikan ke Plugins > Add New > Upload Plugin
3. Pilih file zip yang telah Anda unduh dan klik "Install Now"
4. Setelah instalasi selesai, klik "Activate Plugin"

## Penggunaan

### 1. Konfigurasi Form

1. Silahkan pilih form yang mau anda edit
2. Klik icon 'gear' dan kemudian klik 'Edit'
![Settings Ninjaform](/plugin/ninjaform-settings.png)

### 2. Mengatur Webhook di NinjaForm

1. Pada Editor Ninjaform, Pilih tab 'Advanced'
2. Pilih Menu 'DripSender'
![Settings Ninjaform Advanced](/plugin/ninjaform-advanced.png)
3. Pada menu 'DripSender', Silahkan isi webhook URL dengan URL dari integrasi dari DripSender
4. Pilih Field yang akan digunakan sebagai NAMA dan juga PHONE
![Settings Ninjaform Webhook](/plugin/ninjaform-webhook.png)
5. Simpan Perubahan dan Publish Ninjaform
6. Voila! Setiap input dari Ninjaform akan terintegrasi dengan sistem DripSender
