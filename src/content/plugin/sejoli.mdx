# Dokumentasi Plugin Dripsender Sejoli

## Deskripsi Plugin

Plugin "Dripsender Sejoli" adalah solusi integrasi antara sistem notifikasi Sejoli dengan layanan Dripsender.id untuk pengiriman pesan WhatsApp. Plugin ini memungkinkan Anda mengirimkan notifikasi WhatsApp secara otomatis kepada pelanggan Anda melalui platform Dripsender.id.

> Link download plugin Dripsender-Sejoli ada di akhir artikel ini.

## Persyaratan Sistem

- WordPress (versi terbaru)
- Plugin Sejoli (terinstall dan aktif)
- Akun Dripsender.id aktif
- API Key dari Dripsender.id

## Langkah-Langkah Instalasi

### 1. Install dan Aktivasi Plugin

Upload, install, dam aktifkan plugin "Dripsender Sejoli" melalui menu Plugins di dashboard WordPress Anda.

### 2. Salin Api Key Dripsender

Buka dashboard akun Dripsender, pilih nomor WhatsApp yang ingin Anda gunakan, salin API-KEY nya.
![Thumbnail Plugin](/plugin/sejoli-api-ds.png)

Setelah instalasi selesai, aktifkan plugin "Dripsender Sejoli" melalui menu Plugins.

### 3. Konfigurasi Plugin

Navigasi ke: Pengaturan Sejoli > Notifikasi > Pengaturan WhatsApp Service

> Pilih "Dripsender.id" sebagai layanan WhatsApp
> ![Thumbnail Plugin](/plugin/sejoli-dashboard.png)

> Masukkan API Key dari akun Dripsender.id Anda
> ![Thumbnail Plugin](/plugin/sejoli-inputkey.png)

> Klik "Simpan Perubahan"

> **⚠️ Catatan Penting:**  
> Pastikan API Key yang Anda masukkan sudah benar dan akun Dripsender.id Anda dalam keadaan aktif. API Key dapat ditemukan di dashboard akun Dripsender.id Anda.

## Verifikasi Instalasi

Untuk memastikan plugin berfungsi dengan baik:

1. Lakukan test pengiriman pesan WhatsApp melalui menu Sejoli
2. Periksa status pengiriman di dashboard Dripsender.id
3. Pastikan pesan diterima oleh nomor tujuan
